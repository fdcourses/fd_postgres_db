Чтобы была возмоность пользоватся клиентам постгреса нужно провести слудующие шаги после установки:
1. зайти в папочку /etc/postgresql/{ВАША ВЕРСИЯ БАЗЫ ДАННЫХ}/main
2. выполнить команду sudo nano pg_hba.conf
3. Во всех местах где прописан peer изменить его на trust
  3.1 для пущего эффекта trust можно поставить на всех соединениях
4. сохранить изменения и перезапусить сервис постгреса командой sudo systemctl restart postgresql

После этого появится возможность пользоватся клиентами постгреса без ошибок.

Изменение пароля пользователя
легко выполняется с посошью встроенного клиента psql
1. войти от имени postgres с помощью команды psql -U postgres
2. сменить пароль можно командой \password postgres
3. ввести новый пароль 2 раза (чтобы не забыть можете его оставить postgres)
